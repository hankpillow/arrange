Arranging your view.

Arranging objects is a simple task but boring and very repetitve.
Usually the majority of view's code is about alignment and positioning objects on the screen.
Thinking on a way to avoid this repetition since the last year I've been coding the Arrange.
It's a simple static class with only one objective: arrange things on the screen!

The main methods are listed below:

  • toLeft
  • byLeft
  • toRight
  • byRight
  • toTop
  • byTop
  • toBottom
  • byBottom
  • center (wrapper centerX and centerY)
  • centerX
  • centerY
  • round (rounds x and y)
  • vGrid (creates a vertical grid)
  • hGrid (creates an horizontal grid)
  • byDepth
  • simulate (see above)
  • chain (see above)

How to use it?

place( [box, circle, field] ).toRight( );

What's happening?

  1. "place" is just a wrapper for "new Arrange()"
  2. "box" is a simple sprite
  3. "circle" is a sprite with the register point centered. Note that the script is able to figure the real bounds of this object
  4. "field" is a simple textfield.
Nevertheless to have the same result in a conventional way you'll probably script these lines (considering only this couple of objects): 
circle.x = box.x + box.width + circle.width * 0.5;
field.x = circle.x + circle.width * 0.5;

And only these lines:

place( [stage,box,circle,field] ).centerX().toBottom();

Against all these:

box.x = stage.stageWidth*.5 - box.width*.5;
circle.x = stage.stageWidth*.5;
field.x = stage.stageWidth*.5 - field.width*.5;
circle.y = box.y + box.height + circle.height*.5;
field.y = circle.y + circle.height*.5;

Pretty simple!

Arrange supports also an extra parameter which is a simple Object to provide special setting such as paddings, fixed width/height and etc (see ArrangeProperties):

place( [box,circle,field]).toRight({padding_x:5});
// or just {x:5}
Another great features are the "grid methods" (hGrid and vGrid) what place your objects in a grid format and returns a Grid class to easily manages every single item.

Vertical grid:

var grid : Grid = place(list).vGrid( 3, {padding_x:2, padding_y:2} );
// list is an array with many sprites
// 3 is the number of columns

Horizontal grid:

var grid : Grid = place(list).hGrid( 3, {padding_x:2, padding_y:2} );
// list is an array with many sprites
// 3 is the number of rows
