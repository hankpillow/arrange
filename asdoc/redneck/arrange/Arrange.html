simulate()method 
public function simulate(value:Boolean = true):Arrange

Simulate means: Arrange will perform every method you'll call into arrangeList but will otchange any position for your real objects. To get the result for this simulation you must check the list content and use only the targetX and targetY values. The idea of simulating an Arrage is that you'll be able to change the position when you want and perform any tween engine you prefer.

Parameters

value:Boolean (default = true) — Boolean true starts smulating and false stop.

Returns
